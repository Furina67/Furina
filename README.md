# Setup & Installation

1. Clone the Repository
git clone https://github.com/your-username/furina.git
cd furina

2. Install Dependencies
- pip install -r requirements.txt

3. Add Your Bot Token

TOKEN = "YOUR_BOT_TOKEN"
bot.run(TOKEN)

4. Start the Bot
- python bot.py
